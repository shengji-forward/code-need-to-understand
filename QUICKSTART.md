# Quick Start Guide

## Getting Started

1. **Install dependencies** (one-time setup):
```bash
cd code-need-to-understand
npm install
```

2. **Run any practice file**:
```bash
npx tsx 00-fundamentals/practice/01-variables-and-types.ts
```

3. **Work on exercises**:
```bash
# Open exercise file in your editor
code 00-fundamentals/exercises/exercise-01-basic-syntax.ts

# Run to test your solution
npx tsx 00-fundamentals/exercises/exercise-01-basic-syntax.ts
```

4. **Compare with solution**:
```bash
# View solution file
code 00-fundamentals/exercises/exercise-01-basic-syntax-solution.ts
```
